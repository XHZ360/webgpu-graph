# schema

`schema` 包是 `webgpu-graph` 的核心数据模型层，负责 Schema 类型定义、校验、工厂、摘要与可视化基础能力。

## 文档

- `docs/SCHEMA_MODEL.md` — 静态数据模型与字段约束（Buffer / Binding / Shader / Pipeline / Pass / RenderGraph / Simulation）
- `docs/SCHEMA_RUNTIME.md` — 运行时规范（Builder / Validator / Factory / 执行流程）
- `docs/PROJECT_OVERVIEW.md` — 项目概览与设计目标
- `docs/IMPLEMENTATION_PLAN.md` — 实施顺序与维护建议

> 除非另有说明，文档中的接口均属于目标规范，实际代码仍处于早期阶段。
